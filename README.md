[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## 安装依赖
在运行之前需要先安装node

下载地址 http://nodejs.cn/

下载第三方依赖包
```bash 
$ npm install
```

## 运行程序

在运行程序之前需要连接数据库，只需要配置数据的端口号集合在根目录下 ormconfig.json 文件中配置

```bash
$ npm run start:dev
```

### 填充数据库信息
```bash
$ npm run seed
```
